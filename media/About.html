Primerize

Primerize (previously known as NA_thermo), is an in-house set of MATLAB scripts developed by the Das Lab. It is a small archive of primer design and nucleic acid thermodynamic scripts used by the Das lab at Stanford for high-throughput RNA synthesis and design.

Primerize Server

Primerize Server is an online version of Primerize. It provides the basic functionallity of primer design for bridging PCR.
