What is Eterna?

Eterna is a massively mutiplier game and the first experimental "cloud" lab where players can test their own scientific hypotheses. By playing EteRNA, you will participate in creating the first large-scale library of synthetic RNA designs. Your efforts will help reveal new principles for designing RNA-based switches and nanomachines.

What is an Eterna Score?

Synthetic RNA designs submitted by players are given a score of how well their primary sequence folds into the secondary structure. A score of 100 the maximum score a design can obtain, which means that the primary sequence folded into a structure that is experimentally the indistinguishable from the target structure.

What is EternaDrone?

EternaDrone takes the wealth of data already present Eterna and attempts to harnass or building models for predicting Eterna Scores base using information about the primary and target secondary structure of RNAs. For each construct run through Eterna (~7000) collected so far, features are extracted for both the primary sequence and secondary structure and are fit to the eterna score using a Random Forest regression model. You can use this model to predict the Eterna Score for your own Sequence/Secondary structure here. It is also possible to submit your own model using these features here. It might be possible to come up with very simple rules that have great predictive power that have not been found with machine learning.

What is a Feature?

A feature is loosely defined as any value that can be calculated based on sequence/secondary structure of a construct that can be correlated with its final SHAPE score.

No one parameter is likely to highly correlate directly with the SHAPE score but by using a combination of many it is possible to build an empirical model to help predict the success of a sequence to form into its target secondary structure.

An few example features: the number of Guanine bases in the sequence, the number of basepairs in the secondary structure, etc. See the full list of implemented features.
