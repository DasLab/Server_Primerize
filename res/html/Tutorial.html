Section #2: Result Retrieval

Once submitted, a modal window will be displayed on the web page, showing the JOB_ID :


Primerize is running...

Your JOB_ID is: 0123456789abcdef


Your query is being processed. Usually, the calculation is finished within 30 seconds. Depending on the input sequence length and complexity, the run may take longer.

You can close the browser and retrieve the result later using the above unique JOB_ID. The cached result expires after 7 days.


When the calculation is completed, the result page will automatically be shown. The run results are retrievable through the   Home   page with the unique 16-digit hexadecimal JOB_ID.


Result retrieval

 

Previous result is retrievable by its unique JOB_ID before expiration (7 days).

Before the calculation is finished, the result page will indicate that the query is being processed. Once finished, the actual results will replace this temporary page:


Primerize is running...


Your query is being processed. Usually, the calculation is finished within 30 seconds. Depending on the input sequence length and complexity, the run may take longer.

You can close the browser and retrieve the result later using the above unique JOB_ID. The cached result expires after 7 days.

For long inputs, we encourage the user to   Download   the source code and run locally, which enables more options. If the server encounters an internal error while processing the run, the following page will be shown.


Primerize has difficulty in your query...


Primerize encountered an internal error while processing your query. Sorry for the inconvenience.

We will investigate and fix the problem.

Section #6: PCR Assembly Protocol

Experimental Setup for PCR Assembly of Template

PCR Recipe

Using P4P6_2HP as an example, mix the following components in a PCR tube:

Concentration Components Volume (μl)
5xHF Buffer10.0
10 mMdNTPs (A,T,C,G)1.0
100xPhusion Polymerase1.0
1 μM Primer 2R, 3F, 4R, 5F 2.0 (each)
100 μM Primer 1F, 6R 2.0 (each)
ddH2O26.0
Total50.0

PCR Conditions


Steps Time Temperature
Denaturation30 s98 °C
Denaturation10 s98 °C
Annealing30 s60 °C*
Extension30 s72 °C
         Repeat 30 cycles
Final Extension10 min72 °C
Note:

1. For terminal primers (1F, 6R), use 100 μM. For internal primers (2R, 3F, 4R, 5F), dilute to 1 μM. Add 2.0 μl of each primer to mixture.

2. Use stock of 10 mM dNTPs, which has 10 mM of each dATP, dTTP, dCTP, dGTP mixed together.

3. Choose annealing temperature wisely. We recommend to use the same as min_TM by Primerize design, which is usually between 60-64 °C.

4. Check PCR product on 4% agarose gel. If assembly is unsuccessful with shorter mis-priming products, we suggest try raising the annealing temperature to reduce mis-priming. Alternatively, splitting the assembly into separate sub-pools (i.e. primer 1-4 and primer 5-6) and do an additional round of full assembly.
